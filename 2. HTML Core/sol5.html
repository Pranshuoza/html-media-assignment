HTML
The standard markup language for creating web pages. It structures content, defines elements, and forms the skeleton of a website.
CSS
Styles the appearance of HTML elements on a web page. It controls layout, colors, fonts, and other visual aspects to enhance user experience.
JavaScript
A programming language that enables dynamic behavior on web pages. Used for client-side scripting, it allows interactive elements like animations, form validation, and real-time updates.
React
A JavaScript library for building user interfaces, especially single-page applications. It allows developers to create reusable UI components and manage state efficiently.
Node.js
A JavaScript runtime that lets developers execute JavaScript on the server side. Used for creating scalable backend services, APIs, and real-time applications.
Express.js
A web application framework for Node.js, designed for building APIs and web applications. It simplifies routing, request handling, and middleware integration.
MongoDB
A NoSQL database that stores data in flexible, JSON-like documents. Ideal for handling large volumes of data with dynamic schemas, often used in modern web applications.
SQL
A relational database language used to manage and query structured data. Technologies like MySQL and PostgreSQL are common choices for storing and retrieving data in web apps.
Git
A version control system used to track and manage code changes. It enables collaboration, versioning, and backup of code across a development team.
GitHub
A platform for hosting and managing Git repositories. It provides version control, collaboration tools, and integrates with CI/CD pipelines for seamless deployment.
