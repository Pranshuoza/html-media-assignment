1. <html> - The root element of an HTML document. It wraps all the other elements on the page.
  2. <head> - Contains metadata about the document, such as the title, character set, and links to stylesheets and scripts. It does not display on the page.
  3. <title> - Sets the title of the webpage, which appears on the browser tab and is used by search engines.
  4. <body> - Contains all the visible content of the web page, such as text, images, links, and other elements.
  5. <h1> to <h6> - Heading tags, with <h1> being the largest and most important, and <h6> the smallest. These tags help structure content and improve accessibility.
  6. <p> - Paragraph tag, used to group blocks of text into paragraphs.
  7. <a> - Anchor tag, used to create hyperlinks to other pages, locations, or files. It uses the "href" attribute to specify the link destination.
  8. <img> - Embeds images into the page. It requires a "src" attribute for the image source and an "alt" attribute for accessibility.
  9. <ul> - Unordered list tag, used to create a bulleted list. Each list item is nested within an <li> tag.
  10. <ol> - Ordered list tag, used to create a numbered list. Each list item is nested within an <li> tag.
  11. <li> - List item tag, used within <ul> or <ol> to define individual items in a list.
  12. <div> - Division tag, used to group content into block-level sections, often for styling or layout purposes.
  13. <span> - Inline container tag, used to style or manipulate specific portions of text or content inline, without breaking the flow.
  14. <form> - Contains interactive controls, like input fields, buttons, and checkboxes, for user data submission.
  15. <input> - Creates various types of input fields within a form, such as text boxes, checkboxes, and radio buttons.
  16. <button> - Creates a clickable button, often used for submitting forms or triggering actions.
  17. <table> - Defines a table structure for displaying tabular data. It contains <tr> (rows), <td> (data cells), and <th> (header cells).
  18. <script> - Used to embed or reference JavaScript code in an HTML document for interactive functionality.
  19. <link> - Used to link external resources like CSS files. It is placed within the <head> and does not display on the page.
  20. <meta> - Provides metadata about the document, such as character encoding, author, and viewport settings for responsive design.
